# Clothing Store Website

## Description
A responsive clothing store website built using HTML, CSS, Bootstrap and JavaScript. 
The project includes login page, user profile, product catalogue, collections and contact sections.

## Features
- Responsive design
- Product catalogue UI
- User login and profile pages
- Contact form layout
- Bootstrap styling

## Tech Stack
HTML, CSS, Bootstrap, JavaScript

## How to Run
1. Download the repository
2. Open Profile.html in a browser

## Future Improvements
- Backend integration with Django
- Real add-to-cart functionality
- Database integration for products and users
